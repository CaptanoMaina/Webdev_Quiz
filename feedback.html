1. Which of the following is the correct format for the HTML5 doctype?

Feedback

The doctype declaraton must be the very frst thing in your HTML document, before the tag. It is not an HTML tag; it is an instructon to the web browser about what version of HTML the page is writen in.

2. Which of the following examples is the correct way to markup a paragraph?

Feedback

<p1> </p1> is the correct format. <paragraph> and <p1> do not exist within HTML.

3. Which of the following examples would you expect to usually render as bold text?

Feedback

<strong> is a phrase tag that defines important text. This is usually rendered as bold text in the browser. Not to be confused with the <b> tag, which is used to draw attention to without indicating additional importance. <bold> does not exist within HTML.

4. The following is a directory structure of a website with the domain meme.com:
The root directory has index.html, contacts.html and a folder called posts. The posts directory has post1.html.
The web designer intends to add a link in post1.html that takes the user to contacts.html. Which of the following links will not achieve this?

Feedback

A. This is a relative link that links direct the browser to resources based on their current location/directory. The .. in the link tells the browser to move up one directory. As indicated post1.html is in the posts directory which is contained in the root directory. Hence using this relative link the browser is able to correctly access the contacts.html file that's in the root directory.

B. This is a type of relative link known as root relative link. With this link, the browser will append the root domain, in this case meme.com, to the front of the link to get the full url to the resource. Hence this will be meme.com/contacts.html. As per the directory structure indicated, this is resource exists.

C. This will not access contacts.html. It's a relative link similar to the one in option A but here, it is pointing to a resource called contacts.html in the current directory. The current file, post1.html, is contained in the posts directory. The posts directory does not contain a contacts.html file, hence the link will not return the contacts page.

D. This is an absolute link. This link refers to the resource by including the full domain path. Despite retrieving contacts.html page, its use is discouraged when a webpage is referring to resources on the same website. Relative links are preferred since any changes to the domain will not require any code to be changed. Absolute links are usually used to access resources on a different website

5. The following symbols have special meaning in HTML and hence have to be encoded in order for the browser to display the symbol.
Which one does not need to be encoded for it to be displayed?

Feedback

A. The < sign is used to indicated the start of a HTML tag (opening or closing). Hence it has to be encoded in order for the browser to display it as is. Its encoding is <

B. The ; sign is used to indicate the end of a character entity encoding. Despite this, it can be written as is because it only gets meaning when a & is encountered. That notwithstanding, its encoding is ;

C. The > sign is used to indicate the end of a HTML tag (opening or closing). Hence it has to be encoded in order for the browser to display it as is. Its encoding is >

D. The & sign is used to indicate the start of a character entity encoding. Therefore, in order for it to be printed as is, it has to be encoded. Its encoding is &
